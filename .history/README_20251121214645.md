## Informazioni utili per sviluppo e manutenzione üß≠

### Struttura del progetto
- / ‚Äî file HTML principali (index.html, ecc.).  
- css/, js/, images/ ‚Äî asset frontend.  
- backend2.py ‚Äî logica server-side (eseguire separatamente).  
- requirements.txt ‚Äî dipendenze Python (se presente).

### Ambiente di sviluppo (consigli)
1. Creare un virtualenv:
    ```
    python -m venv .venv
    source .venv/bin/activate   # Windows: .venv\Scripts\activate
    pip install -r requirements.txt
    ```
2. Adattare i comandi di avvio al framework usato (Flask o FastAPI):
    - Flask:
      ```
      export FLASK_APP=backend2.py
      flask run
      ```
    - FastAPI (uvicorn):
      ```
      uvicorn backend2:app --reload
      ```
    Modificare i nomi/variabili se diversi.

### Configurazione frontend
- Tenere l'URL base dell'API in una variabile centrale (es. config.js o const API_BASE = "...") per poterla cambiare facilmente tra ambiente locale e produzione.
- Abilitare CORS nel backend se il frontend e il backend sono su domini diversi.
